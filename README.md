# AI 解剖教学 · 腹直肌鞘与弓状线 3D 交互可视化

系统解剖学课堂 AI 辅助教学工具。同一个 WiFi 下师生可同时操作，也可部署到 GitHub Pages 让学生随时随地访问。

## 🚀 在线访问

部署后，学生打开以下网址即可使用（无需安装、无需登录）：

```
https://<你的用户名>.github.io/<仓库名>/
```

## 🧠 教学内容

腹直肌鞘前后层的腱膜构成，弓状线上下腹前壁肌群的层次关系。

| 区域 | 鞘前层 | 鞘后层 |
|------|--------|--------|
| 弓状线以上 | 腹外斜肌腱膜 + 腹内斜肌腱膜前层 | 腹内斜肌腱膜后层 + 腹横肌腱膜 |
| 弓状线以下 | 腹外斜肌腱膜 + 腹内斜肌腱膜 + 腹横肌腱膜 | 缺如（腹直肌直接贴腹横筋膜） |

## 🎮 操作方式

| 方式 | 说明 |
|------|------|
| 🖱 鼠标拖拽 | 旋转 3D 模型 |
| 🔍 滚轮 | 缩放 |
| 🎤 语音指令（笔记本） | 说"滑动截面""弓状线以上""讲解""临床"等 |
| ⌨ 文字输入 | 底部输入框打字，回车执行 |
| 🔘 快捷按钮 | 完整 / 滑动截面 / 弓上线 / 弓下线 / 截面 / 标注 / 复位 |
| 📱 手机 | 触控拖拽旋转 + 双指缩放 + 按钮操作 |
